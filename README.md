# Электронная библиотека - REST API

Django REST Framework проект для управления электронной библиотекой с книгами и авторами.

## Функциональность

### Для всех пользователей:
- Поиск книг по названию, жанру, автору
- Просмотр информации о книгах и авторах
- Чтение книг и просмотр обложек

### Для администраторов:
- Добавление новых книг и авторов
- Редактирование существующих записей
- Удаление книг и авторов

## Технологии

- Python 3.x
- Django 5.2.8
- Django REST Framework 3.16.1
- Pillow 11.3.0 (работа с изображениями)
- SQLite (база данных)

## Установка и запуск
1. Клонируйте репозиторий:
 `git clone <ваш-репозиторий>
  cd RestApiDjango`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Примените миграции:
   `python manage.py migrate`
4. Запустите сервер
   `python manage.py runserver`
